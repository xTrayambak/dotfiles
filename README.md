# My personal workflow dotfiles
This kind of serves as a backup or in case someone requests a dotfile from me. Enjoy!

# Software
* wezterm: a bLaZinGlY fAsT GPU-accelerated terminal written in Rust
* Hyprland: a visually appealing Wayland (wlroots) compositor with a maintainable C++ codebase
* swaync: a cool notification daemon for wlroots compositors written in Vala
* waybar: a highly customizable status bar written in C++
* swaylock: a highly customizable session locker for wlroots-based compositors
* wofi: a Wayland alternative to Rofi
